Basically i need to do a program that can take care of a car park with 3 floors max and 99 parking spaces max.
This is v1.0 of the project and more languages could be added in the future

English Version:


I - Code it in C#

The necessary C# elements:
- Structures
- Pointers
- Arrays
- External files

1st Stage:
- Create a program that manages 5 parking spaces, whether they are occupied or empty.

2nd Stage:
- Associate an external file that will store the state of the parking spaces.

3rd Stage:
- Improve this program to handle 3 floors with 99 spaces each, i.e., PARK[3][99].

(Note: comment on all program(s).)

Once the 3rd Stage is completed, it will need to be integrated with Processing or Python for the display aspect.
